Kapan saatnya menggunakan `==` dan `===`?

Ketika kita hendak melakukan perbandingan dengan tipe nilai berikut, maka dianjurkan menggunakan `===`.

`NaN undefined null false true "" 0`

Untuk selainnya, tidak apa-apa menggunkan `==`
